# Комментарии к решению

Кажется, не очень хорошо описаны требования к системе, но из того, что я понял:
- Для хранения логов достаточно sequential consistent хранилища, ведь нам не обязательно даже eventually consistent хранения логов...
- Для хранения закоммиченных записей нужно использовать linearizable хранилища.
